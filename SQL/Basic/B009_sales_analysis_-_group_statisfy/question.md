Given a table Sales with columns (seller_id, product_id, buyer_id, sale_date, quantity), 
write a SQL query to find the product_ids that were sold ONLY in the first quarter of 2019 
(i.e., between 2019-01-01 and 2019-03-31 inclusive).

If a product has even a single sale outside this date range, it should be excluded entirely 
from the result.
