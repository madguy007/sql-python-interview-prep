# Find Employees Hired on Weekends

## Problem

You are given an `employees` table.

### Table: employees

| Column | Type |
|------|------|
| id | INT |
| name | VARCHAR |
| hire_date | DATE |

## Task

Write a SQL query to **retrieve all employees who were hired on weekends (Saturday or Sunday).**

## Expected Output

Return all columns for employees whose `hire_date` falls on **Saturday or Sunday**.
