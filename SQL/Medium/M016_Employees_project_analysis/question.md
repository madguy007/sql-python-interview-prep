Employee Projects Analysis

You are given a table:

Table: EmployeeProjects

Columns:
- employee_id (int)
- project_id (int)
- hours (int)

Each row represents the number of hours an employee worked on a project.

An employee is considered "high-performing" if they worked more than the average hours (for that project) in more than 1 project.

Write an SQL query to find all such employee_ids.
