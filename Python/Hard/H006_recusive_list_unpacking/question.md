Flatten Nested List

Easy  
+10 APs  
Solved

Write a Python program to flatten a nested list using recursion.

A nested list may contain integers or other lists inside it.  
Your task is to extract all the numbers and return them in a single flat list.

Example 1:
Input:
[1,[2,3],4,[5,[6,7]],8]

Output:
[1,2,3,4,5,6,7,8]
