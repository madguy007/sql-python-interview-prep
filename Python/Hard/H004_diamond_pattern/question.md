Create a program that prints the following numeric diamond pattern for a given integer n.

Example

Input
5
```

Output
    1    
   123   
  12345  
 1234567 
123456789
 1234567 
  12345  
   123   
    1    
```
Rules
1. The middle row contains numbers from 1 to (2*n - 1)
2. Each row above the middle increases numbers by 2
3. Each row below the middle mirrors the upper half
4. The pattern must remain center aligned
