Easy

Solved

Write a Python program to print the following numeric pattern.

The pattern forms a square with numbers decreasing toward the center.

Example

Input  
5

Output
555555555
544444445
543333345
543222345
543212345
543222345
543333345
544444445
555555555

Explanation

- The outer boundary is filled with the largest number.
- As we move toward the center, the number decreases.
- The smallest number appears at the center of the pattern.
