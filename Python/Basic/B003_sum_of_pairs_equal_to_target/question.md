# Two Sum Unique Pairs

Given a list of integers and a target value, find all **unique pairs** of numbers whose sum equals the target.

Return the pairs without duplicates.

## Example

Input

nums = [2, 4, 3, 5, 6, -2, 4, 7, 8, 9]  
target = 7

Output

[(2,5), (3,4), (-2,9)]

## Rules

- Each pair should sum to the target.
- Do not return duplicate pairs.
- Order inside the pair should not matter.

Example:

(3,4) and (4,3) should be considered the same pair.
