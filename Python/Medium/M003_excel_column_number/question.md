# Excel Sheet Column Number

**Difficulty:** Hard

Given a string `columnTitle` that represents the column title as it appears in an Excel sheet, return its corresponding column number.

Excel columns follow a pattern similar to base-26 numbering using letters:

A -> 1  
B -> 2  
...  
Z -> 26  
AA -> 27  
AB -> 28  

---

## Examples

Input
A

Output
1

Input
AB

Output
28

Input
ZY

Output
701

Input
FXSHRXW

Output
2147483647
