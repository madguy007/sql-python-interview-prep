# Cross Pattern (Easy)

Write a Python program to print a cross pattern (X) for a given number **N**.

A cross pattern forms an **X shape** where:
- One `*` appears on the **main diagonal**
- One `*` appears on the **secondary diagonal**

## Input
An integer `N` representing the number of rows.

## Example
 N = 5
```

* - - - *
- * - * -
- - * - -
- * - * -
* - - - *

```
