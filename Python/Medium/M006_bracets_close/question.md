Balanced Parentheses

Easy  
+10 APs  
Solved

You are given a string containing only ( and ).

Check whether the parentheses are balanced.

A string is balanced if:

Every ( has a matching )

A ) never appears before its matching (

Return True if the parentheses are balanced, otherwise return False.

Examples

Input
"(())()"

Output
True


Input
"(()))("

Output
False


Input
")("

Output
False
