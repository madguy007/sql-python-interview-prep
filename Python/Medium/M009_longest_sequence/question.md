Given an array of integers nums, return the length of the longest consecutive sequence that can be formed.

A consecutive sequence means:
each next number is exactly 1 greater than previous number.

The numbers do NOT need to appear in order in the original array.

Example:

Input:
nums = [100, 4, 200, 1, 3, 2]

Output:
4

Explanation:
Longest consecutive sequence:
[1, 2, 3, 4]

Length = 4
